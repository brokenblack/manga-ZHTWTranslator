  遊戲即時翻譯工具  ─  使用說明
  日文／韓文 OCR + AI 翻譯 + Anki 單字卡匯出
═══════════════════════════════════════════════════════


【啟動程式】

  雙擊「翻譯工具.exe」即可。
  首次啟動約需 5–10 秒（要解壓內建模組）。


───────────────────────────────────────────────────────
【第一次使用：申請 API Key（必要）】
───────────────────────────────────────────────────────

本程式使用 AI 翻譯，需要至少一組 API Key。
推薦從免費的 Groq 開始：

  1. 開啟 https://console.groq.com 註冊帳號
  2. 左側選單「API Keys」→「Create API Key」
  3. 複製產生的 key（gsk_... 開頭）
  4. 回到本程式，點右上角「⚙️ 翻譯設定」
  5. 切到「翻譯 API」分頁
  6. 在「Groq」欄位貼上 key
  7. 按「儲存」

其他可選 API（依需求自行申請）：

  ★ Groq         https://console.groq.com           免費，速度最快，推薦入門
    Gemini       https://aistudio.google.com         免費額度大，品質佳
    Claude       https://console.anthropic.com       付費，解說最詳細
    OpenRouter   https://openrouter.ai/keys          可選多種模型，部分免費
    Ollama       https://ollama.com                  本地執行，完全離線


───────────────────────────────────────────────────────
【翻譯模型優先級（自動 fallback）】
───────────────────────────────────────────────────────

可同時設定多個 API，依優先順序自動 fallback：

  例：[Gemini, Ollama]
    → 優先用 Gemini（網路 + 免費）
    → 沒網路時自動切到 Ollama（本地）

設定方式：
  ⚙️ 翻譯設定 → 翻譯 API → 點「翻譯使用模型」清單
  用 ↑ ↓ 調順序、⊕ 加入、⊖ 移除


───────────────────────────────────────────────────────
【翻譯遊戲畫面】
───────────────────────────────────────────────────────

方法一：快捷鍵（推薦）

  按  Ctrl + Alt + Q  →  拖曳框選遊戲文字  →  自動翻譯
  按  ESC             →  取消選取

方法二：點擊主視窗

  「📷 OCR 截圖」分頁 → 點「✂️ 選取區域並翻譯」


───────────────────────────────────────────────────────
【Textractor 整合（VN 視覺小說玩家）】
───────────────────────────────────────────────────────

如果你已經在用 Textractor 攔截遊戲文字：

  1. Textractor 內啟用「Extensions → Copy to Clipboard」
  2. 本程式切到「🔗 Textractor」分頁
  3. 點「▶ 開始監聽」或按 Ctrl + Alt + T
  4. 遊戲文字自動翻譯
  5. 按 ESC 或再按一次熱鍵停止監聽


───────────────────────────────────────────────────────
【匯出單字到 Anki（選用）】
───────────────────────────────────────────────────────

需先安裝：

  1. Anki 桌面版：https://apps.ankiweb.net
  2. AnkiConnect 外掛：
     在 Anki 內 → 工具 → 附加元件 → 取得 → 輸入：2055492159

之後每次翻譯結果旁會出現：

  「📝 加入句子卡」  ─ 整句加入卡片
  「🎴 加入挖空卡」  ─ 點選詞元做 Cloze 卡（推薦學習新單字）

歷史紀錄側欄可勾選多筆批量匯出。


───────────────────────────────────────────────────────
【真人發音（選用）】
───────────────────────────────────────────────────────

製卡時可加入真人發音（需 Forvo API Key）：

  1. https://api.forvo.com 申請 API Key
  2. ⚙️ 翻譯設定 → Anki → 貼入 Forvo API Key
  3. 之後每張卡會自動帶 Forvo 真人發音


───────────────────────────────────────────────────────
【設定總覽】
───────────────────────────────────────────────────────

⚙️ 翻譯設定：

  • OCR：選 manga-ocr（日文漫畫專用）或 EasyOCR（日韓多語）
  • 翻譯 API：填 API Key、設定優先順序
  • Textractor：剪貼簿過濾規則
  • Prompt：自訂 AI 翻譯指令（進階）
  • 一般：截圖快捷鍵、監聽快捷鍵、開機自動啟動

🃏 Anki 設定：

  • Deck（牌組）名稱
  • Note Type（卡片樣板）
  • 欄位對應


───────────────────────────────────────────────────────
【全域快捷鍵】
───────────────────────────────────────────────────────

  Ctrl + Alt + Q   截圖選取翻譯（任何時候，遊戲全螢幕也能用）
  Ctrl + Alt + T   切換 Textractor 剪貼簿監聽
  Ctrl + Alt + W   從系統匣叫出主視窗（縮到背景後快速喚回）
  ESC              取消截圖選取 / 取消 Textractor 監聽

截圖與監聽兩組熱鍵可在「⚙️ 翻譯設定 → 一般」自訂；
叫出視窗的 Ctrl + Alt + W 為固定快捷鍵。


───────────────────────────────────────────────────────
【常見問題】
───────────────────────────────────────────────────────

Q：Windows Defender 說有病毒，無法執行？
A：個人開發工具沒有商業簽章造成的誤報。處理方式：
   1. 開啟「Windows 安全性」
   2. 「病毒與威脅防護」→「保護歷程記錄」
   3. 找到本程式 →「動作」→「允許」
   或將整個資料夾加入「排除項目」。

Q：截圖快捷鍵在某些遊戲沒反應？
A：如果遊戲是用「以系統管理員身分執行」開啟的，本程式也必須
   用系統管理員執行才能攔截快捷鍵。請對「翻譯工具.exe」
   點滑鼠右鍵 →「以系統管理員身分執行」。

Q：第一次截圖很慢？
A：首次使用會自動下載 OCR 模型（manga-ocr 約 400MB，
   EasyOCR 約 300MB／語言）到使用者目錄，有進度顯示。
   下載完成後即可離線使用 OCR。

Q：翻譯一直失敗？
A：依序檢查：
   1. ⚙️ 翻譯設定 → 翻譯 API：是否有填 API Key？
   2. 網路是否正常？
   3. 該 API 是否額度用盡？（可以加入第二個 provider 做 fallback）
   4. 狀態列會顯示錯誤原因

Q：Anki 卡片新增失敗？
A：確認：
   1. Anki 桌面版正在執行
   2. 已安裝 AnkiConnect 外掛（重啟 Anki 後生效）
   3. 🃏 Anki 設定中 Deck/Note Type 名稱與你 Anki 內的一致

Q：可以完全離線使用嗎？
A：可以。安裝 Ollama (https://ollama.com)，下載一個模型
   （建議 gemma2:9b 或 qwen2.5:14b），在「翻譯 API」設定
   優先級為 [Ollama] 即可。OCR 部分本來就是離線的。

Q：能改翻譯風格嗎？
A：可以。⚙️ 翻譯設定 →「Prompt」分頁，可自訂 AI 指令。
   按「↺ 還原預設」可還原。

Q：設定檔在哪？要備份的話備份哪個檔？
A：與「翻譯工具.exe」同一資料夾下的：
     • config.json   ─ 所有設定（含 API Key）
     • history.json  ─ 翻譯歷史
   只要備份這兩個檔即可。


───────────────────────────────────────────────────────
【系統需求】
───────────────────────────────────────────────────────

  • Windows 10 或 11（64 位元）
  • 記憶體：建議 4 GB 以上
  • 硬碟：解壓後約 1.5 GB（含 OCR 模型約 2.5 GB）
  • 網路：使用線上 AI 翻譯時需要
